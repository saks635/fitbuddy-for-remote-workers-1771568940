## About FitBuddy

FitBuddy is an innovative AI-powered virtual fitness coach dedicated to combating the sedentary lifestyle and stress often associated with remote work. It offers a comprehensive platform for tracking daily workouts, nutrition, and mental health check-ins, delivering personalized training plans, AI-guided meditation, and gamification to keep you motivated and engaged.

## Features

*   **AI-Powered Personalization:** Tailored workout and nutrition plans based on your data.
*   **Holistic Wellness Tracking:** Monitor physical activity, dietary habits, and mental well-being.
*   **AI-Guided Meditation:** Stress reduction and mindfulness exercises.
*   **Gamification & Motivation:** Earn rewards, compete with friends, and stay engaged.
*   **Seamless Integration:** Designed for tech-savvy remote workers, accessible anywhere.

## Target Audience

Remote employees aged 22-45, tech-savvy, and interested in holistic wellness. Individuals seeking to improve their work-life balance, reduce stress, and maintain a healthy lifestyle while working remotely.

## Website Structure

The website consists of the following pages:

*   `index.html`: The main landing page, introducing FitBuddy and its core benefits.
*   `about.html`: Details about FitBuddy's mission, vision, and the problem it solves.
*   `pricing.html`: Information on subscription plans and features offered.
*   `contact.html`: A page for inquiries, support, and getting in touch.
*   `styles.css`: Centralized stylesheet for consistent branding and design across all pages.
*   `script.js`: Shared JavaScript for interactive elements (e.g., mobile navigation, simple animations).

## Technologies Used

*   **HTML5:** For structuring the web content.
*   **CSS3:** For styling and layout, ensuring a responsive and modern design.
*   **JavaScript (ES6+):** For interactive elements and dynamic content.
*   **Google Fonts:** Poppins and Inter for modern typography.
*   **Font Awesome:** For icons.

## How to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone <repository-url> # Replace with your actual repository URL if hosted
    cd fitbuddy-website
    ```
2.  **Open the files:**
    Simply open any of the `.html` files (e.g., `index.html`) in your web browser. All assets are linked relatively, so it should work out of the box.
